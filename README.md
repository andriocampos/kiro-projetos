## 🧭 Como usar este repositório

1. Cada módulo tem seu próprio `README.md` com índice e explicações
2. Dentro de cada módulo, a pasta `docs/` contém teoria detalhada
3. Os arquivos de código (`.tf`, `.yaml`, etc.) contêm comentários explicativos
4. Siga a ordem sugerida no índice de cada módulo
